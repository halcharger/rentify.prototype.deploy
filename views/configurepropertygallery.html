Galleries

Users should be able to manage (add / edit / delete) multiple galleries.

Galleries are assigned a name and longer description as well as a default image inside of the gallery.

Images are uploaded to galleries and can be assigned a caption. One image per gallery is marked as the default or primary image for that gallery

Each individual image has a unique identifier which can be used when referencing images in custom page markup.
